# Тестовое задание — Решение

## Структура
- `block1_probability/` — задачи по теории вероятности
- `block2_python/` — алгоритмические задачи на Python
- `block3_sql/` — SQL-запросы
- `block4_analytics/` — вопросы по A/B-тестированию и статистике
- `block5_stats/` — расчёт корреляции

## Как запустить
Все Python-скрипты можно запускать независимо. Для SQL-запросов требуется СУБД с поддержкой оконных функций (PostgreSQL, MSSQL).
